## Installation

Pasos para instalar el proyecto:

```bash
  git clone https://github.com/SantiSarmiento/global-think.git
  npm install
  cd ios && pod install


  npm run ios
  npm run android
```
    
Versiones utilizadas durante la creación del proyecto:

Node v21.5.0

NPM 10.2.4
